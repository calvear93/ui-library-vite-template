# useCounter

Counter state with an `increment` action.

```ts
const [count, increment] = useCounter(initialValue);
```

- **`initialValue`** (`number`, default `0`) — the starting value.
- **Returns** a readonly `[count, increment]` tuple.

## Live demo

export const CounterDemo = () => {
	const [count, increment] = useCounter();
	return <Button text={`Count: ${count}`} onClick={increment} />;
};
