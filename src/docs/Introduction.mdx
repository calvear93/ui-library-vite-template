# UI Kit

A React component library template, built and bundled with Vite library mode.

- **Atomic Design** — components are organized as atoms → molecules → organisms.
- **Vite 8 library mode** — tree-shakeable ESM output with per-component CSS and generated `.d.ts`.
- **UnoCSS** — style with utilities via `@apply`; the used utilities compile into each component's CSS.
- **Storybook 10** — every component ships a story with autodocs.
- **Tested** — Vitest + React Testing Library, with Stryker mutation testing.

Explore the components in the sidebar (Atoms, Molecules, Organisms), then replace them with your own.
