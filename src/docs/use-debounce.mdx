# useDebounceState

Debounced state — `value` updates only after input stops for `interval` ms.

```ts
const [value, setValue] = useDebounceState(initialState, interval);
```

- **`initialState`** (`S`, optional) — the initial value.
- **`interval`** (`number`, default `200`) — debounce delay in milliseconds.
- **Returns** a readonly `[value, setValue]` tuple.

## Live demo

export const DebounceDemo = () => {
	const [value, setValue] = useDebounceState('', 400);
	return (
		<div style={{ display: 'flex', flexDirection: 'column', gap: '0.5rem' }}>
			<Input
				placeholder="Type…"
				onChange={(event) => setValue(event.target.value)}
			/>
			<span>Debounced (400 ms): {value}</span>
		</div>
	);
};
